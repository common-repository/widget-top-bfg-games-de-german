== Description ==

Diese einfache Plugin ist ein Widget, das eine Liste der Top-Spiele aus BigFish auf Ihrem widgetized Seitenleiste zeigen nur in deutscher Sprache (.de).
Es ist sehr brauchbar, wenn Sie eine Website fur die verschiedenen Sprachen zu erstellen und wollen alle Spitzen von Spielen fur verschiedene Sprachen verwenden mochten.

ENGLISH:
This simple plugin is a widget that displays a list of top casual games from BigFish on your widgetized sidebar on German language only (.de).
It's very useable if you want to create one site for different languages and want to use all tops of games for different languages.

== Installation ==

1. Upload Widget Ordner enthalt 'widget-top-bfg-games-DE.php' die '/wp-content/plugins/'.
2. Aktivieren Sie das Plugin im 'Plugins' Menu von WordPress.
3. Verwenden Sie Ihre 'Presentation'/'Sidebar Widgets' Abschnitt, um das Widget zu platzieren und zu konfigurieren.

ENGLISH:
1. Upload widget folder contains 'widget-top-bfg-games-DE.php' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use your 'Presentation'/'Sidebar Widgets' section to place the widget and to configure it.

Siehe die Arbeiterklasse Plugin unter http://www.casualplayground.com/


== Configuration ==

* Widget Titel: Der Titel des Widgets
* Anzahl der Spiele auf Display: Anzahl der Top-Spiele, die Sie gerne angezeigt bekommen mochten (1-10)
* Spiele Typs angezeigt: Spiele-Typen die Sie angezeigt (PC-Spiele / Online Spiele) ware
* Text und Bilder ausrichten: Ausrichten der Spiele-Liste mochtest angezeigt (Thema default / Links / Mitte / rechts)
* Liste anzeigen Bullets: Ist mussen in Listen zeigen (Nutzung Thema style)
* Show Spiele Position: Ist brauchen, um Spiele Position im Top-Liste zeigen
* Show Spiele Images: Ist brauchen, um Spiele Bilder neben Titel des Spiels zeigen
* Bilder Typ: Bilder Typ Sie angezeigt bekommen mochten (kleine 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Games Text: Ist braucht, um Spiele Text neben Titel des Spiels und Bild zeigen
* Was Oben: Was Above mochten Sie angezeigt werden - Text oder Bild
* Ihre BigFishGames login: Sie konnen dieses Feld leer lassen, Widget kann ohne Anmeldung funktioniert, oder verwenden Sie richtigen Login.
* Ihre BigFishGames Client-Kanal: Domain Name wird standardma?ig verwendet werden. Lassen Sie dieses Feld leer ist, nicht zu verwenden Kanale. Alpha nur numerisch.

ENGLISH:
* Widget title: the title of the widget
* Number of Games to Display: Number of top games you would like to be displayed (1-10)
* Games Type to Display: Games Types you would like to be displayed (PC games / Online Games)
* Text and Images Align: Align of games list you would like to be displayed (theme default / left / center / right)
* Show List Bullets: Is need to show list bullets (use theme style)
* Show Games Position: Is need to show games position in top list 
* Show Games Images: Is need to show games images in addition to game title
* Images Type: Images Type you would like to be displayed (small 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Games Text: Is need to show games text in addition to game title and image
* What Above: What Above you would like to be displayed - Text or Image
* Your BigFishGames login: You can leave this field empty, widget can work without login, or use correct login.
* Your BigFishGames client channel: Domain name will be used by default. Leave this field empty, to not use channels. Alpha numeric only.
